Instructions on how to compile and execute programs. 

To compile and execute these java files:
1. Download and place all the files in the same directory.

2.For a Windows computer open a command prompt and move to the file location using the cd command.


3. Type "javac "File name".java" into the command prompt line

4. This will use the complier to create the byte code. If it compiles correctly a new file will be created for each with a class type. 

5. Type "java 'File Name'.java into the command prompt line

The program will now run.
